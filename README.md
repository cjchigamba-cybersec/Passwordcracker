# Passwordcracker
Multi-threaded password hash cracker supporting dictionary and brute-force attacks , built for cybersecurity learning and CTF challenges.

A lightweight, multi-threaded password cracker built in Python.Supports MD5, SHA-1, SHA-256, and more. Runs dictionary attacks
from a wordlist with automatic brute-force fallback. Configurablecharset, length range, and thread count via CLI.Built as part of a cybersecurity internship project at Inlighnx Global.
For educational and authorized testing purposes only.
